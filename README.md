# SAYN aksesuar

Azərbaycanın trendy aksesuar, elektronika və stiker mağazasının kataloq veb saytı.

- **Texnologiya:** HTML5, CSS3, Vanilla JavaScript (tək fayl: `index.html`)
- **Funksiyalar:** Məhsul kataloqu, filter/sort, səbət, wishlist, məhsul modalı, mobil responsiv
- **Sifariş:** Instagram üzərindən — saytda yalnız kataloq və səbət/wishlist

## İşə salma

`index.html` faylını brauzerdə açın. Şəkilləri `images/` qovluğuna əlavə edin.
